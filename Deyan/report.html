Introduction

The CSY-1018 Web Development course is all about students learning the basics of HTML and CSS properties which enables them to design different web pages and use different text editor through which they can improve their designing skills even more.Furthermore the students also learn tools like Github where they can store their works and access them whenever they want and are also introduced with Google Material Design and CSS3 properties throughout the teaching period.
The assignment required us to create six web pages which contained our personal details.According to the assignment brief the six pages would be:

  1. Introduction: This page is the Landing page.
  2. About Me: This page contains a short discription about me
  3. Contact: You can know how to contact me through this page
  4. Social: My social links are present in this page
  5. CV: A page with my experience and other CV components
  6. Report: A page with the informatiion of the assignment.
